# Use Case Challenge Prompts

This directory contains prompts for challenging multiple LLMs on high-value use cases enabled by the current LinuxFr dataset MVP.

Each subdirectory contains:

- `prompt.md`: the prompt to send to an LLM;
- one empty `response-*.md` file per LLM provider, used to paste the corresponding answer.

Current response files:

- `response-claude.md`
- `response-chatgpt.md`
- `response-gemini.md`
- `response-microsoft-copilot.md`
- `response-perplexity.md`
- `response-grok.md`
- `response-glm-zai.md`

After collecting responses, use these files as inputs for a synthesis pass.
